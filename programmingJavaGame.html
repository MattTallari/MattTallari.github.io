Project Description

This was a short project for my Sound & Motion class where we were assigned to take and existing game or application and alter it with HTML/CSS/JavaScript. I chose a simple, open source game I found online and added onto it. Most of the original game is the base of the project, I mainly changed the graphics and the JavaScript as well as added sound and how the enemy spawning works.

NOTE: The assets used are from Chrono Trigger, made my Square-Enix and Published by Nintendo for the Super Nintendo.

Tools/Skills Used

  • API: Sublime Text 2
  • Code: HTML/CSS/JavaScript
