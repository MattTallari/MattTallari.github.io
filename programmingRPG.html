Project Description

This project is currently a Work In Progress and not yet complete. Estimated time of completion is unknown but this page will update over time. This began as a summer project between my 2nd and 3rd year at Confederation College in the Interactive media Program. I set out to make a simple overhead, action RPG style game where the user can attack enemies in real-time rather than traditional turn-based style RPGs. In addition, players can level up, accept quests, and more. There is no current story, this project was simply to learn and udnerstand mechanics and concepts in Unity using C# and I hope to continue the project and develope more mechanics.

In addition to using Unity I used an external program called "Tiled" to create the areas/maps and exported them into Unity using another programed titled "Tiled2Unity" which, depending on how you organize and layer your maps in Tiled, will transfer the collision and layers into Unity which is incredibly useful, seeing as Unity does not currently have a good way to map out 2D areas.

In the future, I would like to add:

  • Magic, like a healing spell, projectile, barrier, AoE attack, etc.
  • Stamina bar/MP System
  • A Save and Load feature
  • Inventory, useable/collectible objects
  • More varied enemies ex. bosses and more tricky enemies

Tools/Skills Used

  • Unity v5
  • Tiled & Tiled2Unity
  • Code: C#
  • Game Design
  • Level Design
