Project Description

This project was my first venture into creating a mobile application. I chose to create a timer known as a split program or splitter for users to use and track they're progress. It's more or less meant for gamers, more specifically speedrunners, but can be used by anyone for mstly anything from exercising, cooking, etc.

The screens were created using Adobe Illustrator and then imported into Adobe Flash where I then coded the program and created the hotspots/transitions for the functions of the app. On the home screen there are several options. Start brings the user to timer, where users can start, pause, reset, and split their times. Records is a database for users to store their times and records as well as goals, and settings allows for customizability. Currently, only the timer is fully funtional, while the records page and settings are mostly just for show.
