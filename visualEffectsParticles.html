Project Description

This project includes a song revolving around a person attributed to fire and a phoenix, thus I chose to create a particle effect that is similar to embers from a fire and red feathers. In addition, I wanted to make it seem as though the viewer is laying next to the girl, so in the beginning and end it seems as though the viewers eyes are being slowly opened and closed at the beginning and end using a blur and a vignette.

NOTE: The image and music used in this project are NOT mine. I was given permission to use both the image and music.
